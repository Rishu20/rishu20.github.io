Apart from my education, I have worked as a freelance animated ad creator for 18 months with Fiverr and volunteered as a team lead for technical assistance during the NASA Space Apps Challenge. I have also completed several certifications, including IT Automation with Python and Machine Learning on Google Cloud. In my free time, I enjoy working on various projects, such as a gender detection model and a machine-learning algorithm for malware prediction.

I possess a diverse range of skills that make me an excellent fit for any team. With a strong background in programming languages like Python, C, and JavaScript, I am also experienced in web development using React-JS, HTML, CSS, and SCSS. In addition, I have expertise in Machine Learning, Data Science, and Django, along with experience working in a Linux environment. I have also worked on cloud computing platforms like AWS and GCP.

With these skills, I have completed various projects, including a Gender Detection Model, a Malware Prediction Algorithm, and a Word Cloud app, among others. My commitment to continuous learning and development of my skills makes me an asset to any team.

Projects

Blackarch

BlackArch-Linux is now available for WSL 2 on Windows 10 2004 and later versions. With this, users can access over 2500 penetration testing tools on a Windows machine without the need for a virtual machine. The installation process is straightforward, and users can use the Linux terminal to install and run their favorite tools. This provides a convenient way for users to access BlackArch-Linux on their Windows machines for penetration testing and security research purposes.

Covid Tracking

This project visualize COVID-19 case data using Python, Dash, and Plotly. It provides step-by-step instructions for fetching and organizing daily COVID-19 cases data for India using the Pandas library, and developing a short-term prediction model using linear regression. The tutorial then demonstrates how to create interactive visualizations using Dash and Plotly, allowing users to explore and analyze the data.

Covid Prediction

The COVID-19 prediction model for India fetches daily cases data from CSSEGISandData and uses Pandas to store it in a tidy CSV file. Using linear regression, the project develops a short-term prediction model that uses the "Day" column as the independent variable and the "CumConfirmed" column as the dependent variable. The model's accuracy may be affected by changing testing protocols, vaccination rates, and government policies. Nonetheless, the project is a useful tool for analyzing COVID-19 data and making predictions for India.

WordCloud

A word cloud is a visual representation of text data where the size of each word indicates its frequency or importance. It can be used for exploratory data analysis, text mining, and data visualization. Word clouds summarize text data, identify themes and patterns, visualize sentiment and emotions, and enhance presentations and reports. They provide a quick and intuitive understanding of text data, which can inform decision-making, gain insights, and communicate effectively with different stakeholders.

Malware Prediction

This project aims to develop a machine learning model to predict the likelihood of malware attack based on hardware configuration of a given machine. The results indicate the specific machine configuration most vulnerable to malware. However, due to a lack of recent data, the results cannot provide information on vulnerabilities to newer Windows variants. The algorithm involves data classification, sorting, visualization, and model training. Future work could expand on different datasets and attributes for machine learning algorithms.

NewsSelect

This project uses deep learning models to build a text summarization tool that can accurately generate one or two-sentence summaries for news articles. The dataset is preprocessed, cleaned, and tokenized before being trained using the encoder-decoder architecture with attention. The resulting model is saved and deployed using Django, with real-time articles procured from reputable news portals. Users can read personalized news summaries from their preferred news sources on a web application that features a frontend filter component.
